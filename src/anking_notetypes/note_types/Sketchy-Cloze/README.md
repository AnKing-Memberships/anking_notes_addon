# Sketchy Note Type
This is a Cloze style note type designed by <a href="https://www.ankingmed.com">The AnKing</a> for use in all Sketchy-made decks. 


## SKETCHY COLORS
yellow = `#fef4c4` - used for thumbnail text
royal blue = `#0088FA` - used for extra and hint fields and tags. Also used for cloze
dark navy = `#1b2855` - used for cloze color, decided not to use this
light green = `#d1fedc` - not used for anything
